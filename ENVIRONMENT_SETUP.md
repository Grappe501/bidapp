# Environment setup

## Prerequisites

- [Node.js](https://nodejs.org/) LTS (includes `npm`)

## Install

From the repository root:

```bash
npm install
```

## Development server

```bash
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Production build (local check)

```bash
npm run build
npm run preview
```

## Environment variables

None required for the foundation scaffold. When API and AI layers are added, document keys here and use `.env` / `.env.example` per team practice.
